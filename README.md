# SyntaxWear - Tênis e Sneakers Online

## Descrição

SyntaxWear é um projeto de e-commerce de calçados desenvolvido para demonstrar a criação de uma interface de loja virtual moderna e responsiva. O site apresenta um design limpo e atraente, com foco na experiência do usuário e na apresentação visual dos produtos.

## Funcionalidades

- **Página Inicial Atraente**: Uma seção "hero" com imagem de destaque e botões de chamada para ação.
- **Categorias de Produtos**: Seções dedicadas para diferentes categorias de tênis, como Casual, Esporte, Moderno e Futurista.
- **Grid de Produtos**: Uma grade de produtos que destaca os itens mais populares e coleções especiais.
- **Design Responsivo**: O layout se adapta a diferentes tamanhos de tela, proporcionando uma boa experiência em desktops, tablets e smartphones.
- **Navegação Intuitiva**: Menus de navegação claros no cabeçalho e no rodapé para facilitar o acesso às diferentes seções do site.
- **Rodapé Completo**: Inclui links de navegação, formulário de inscrição para newsletter e ícones de redes sociais.

## Tecnologias Utilizadas

- **HTML5**: Para a estruturação do conteúdo do site.
- **CSS3**: Para a estilização e o design visual, utilizando variáveis para um tema consistente e arquivos organizados por componentes.

## Como executar o projeto
- **Clone o repositório**
```
git clone https://github.com/isabela2903/ecommerce-syntaxwear.git
```
- **Navegue até o diretório do projeto**
```
cd ecommerce-syntaxwear
```
- **Abra o arquivo index.html diretamente no seu navegador de preferência (Google Chrome, Firefox, etc.).**
